Exercise 13 Instructions

  • Open Adobe XD

  • Create a custom size design with the dimensions of w:8 by h:11

  • Create a circle that takes up a large portion of the design

  • Place that circle in the middle of the design

  • Set the border of the circle to 0.05

  • Fill the circle, use #480000 as the hex code

  • Create a triangle that is the same size as the circle

  • Place the triangle so that it is overlapping the bottom of the circle with its top angle.

  • Create a rectangle along the bottom of the design and have it overlap half of the triangle's height.

  • Set the border of the rectangle to 0.05

  • Fill the rectangle, use #060951 as the hex code.

  • Create 2 lines overlapping the circle horizontally.

  • Create two lines overlapping the circle vertically, and running perpendicular to the lines from step 12.

  • Done.
